Backup-Analyzer

Der Backupanalyser kann Backupfiles von Sprechstellen der Firma Commend analysieren.
Die häufigsten Fehler werden automatisch erkannt und können korrigiert werden.
Analysierbare Sicherungsdateien müssen die Dateiendung .zip oder .back besitzen.
